Collections

This is going a reference to collections. This includes lists, dictionaries, sets and tuple data types.

Lists

Lists are a type of data structure in python. Lists are a collection of items/members (words, characters, numbers, objects etc.) |

{% highlight python %} sample_list = ['a', 'b', 'c', 1, 2, 3] {% endhighlight %}

Lists are a collection of members. This collection has an order and that order can be modified. Lists also allow for duplicate items/members. In Python, lists are defined by square brackets.
