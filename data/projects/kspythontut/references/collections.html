Collections

This is a reference to collections. This includes lists, dictionaries, sets and tuple data types.

It is important to choose the correct data type (collection type) that is going to be used. Every different type of collection has different rules and the use of the correct type can increase efficiency and even security.


Lists

Lists are a type of data structure in python. Lists are a collection of items/members (words, characters, numbers, objects etc.)

In Python, lists are defined by square brackets.

{% highlight python %} {% raw %} sample_list = ['a', 'b', 'c', 1, 2, 3] {% endraw %} {% endhighlight %}

A few things to note about lists:

- The list collection has an order and that order can be modified. - Lists allow for duplicate members.

Accessing items

An item is referenced in a list through it's index.

Example

Say we want to access (get) member 'a' within the list. We refer to it by it's index. {% highlight python %} {% raw %} sample_list = ['a', 'b', 'c', 1, 2, 3] //Print 'a' print(sample_list[0]) // Index of 'a' is 0. //Print 'b' print(sample_list[1]) // Index of 'b' is 1. //Print 'c' print(sample_list[2]) // Index of 'c' is 2. //Print '1' print(sample_list[3]) // Index of '1' is 3. //Print '2' print(sample_list[4]) // Index of '2' is 4. //Print '3' print(sample_list[5]) // Index of '3' is 5. {% endraw %} {% endhighlight %}

Negative Indexes

It is possible to have a negative index. When a list member is referenced by negative index, the member at the end is obtained. Negative indexing refers to its members from right to left.

{% highlight python %} {% raw %} sample_list = ['a', 'b', 'c', 1, 2, 3] //Print '3' print(sample_list[-1]) //Print '2' print(sample_list[-2]) //Print '1' print(sample_list[-3]) //Print 'c' print(sample_list[-4]) //Print 'b' print(sample_list[-5]) //Print 'a' print(sample_list[-6]) {% endraw %} {% endhighlight %}
