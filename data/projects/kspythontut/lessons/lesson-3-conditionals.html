Conditionals

If this is true, then this happens/is true.

Small intro

Conditionals are a control structure. Control structures are used to change the flow of the code to execute instructions.

The if/else statements

if condition:
    instruction

The if statement is one of the simplest control structures. Assuming condition is a valid condition, if the condition evaluates to true, then instruction executes. If condition evaluates to false, instruction is skipped over and doesn't execute (if else condition isn't specified/ present)

The else statement is used with the if statement. Else is used to tell the code what to do if condition is False.

Examples

x = 5               # This is known as a variable. If you dont really
                      # get this, please let me know so I could clarify.

if x < 10:          # Evaluates to True because 5 is less than 10.
    print("True")   # This executes

if x > 10:          # Evaluates to False. 5 is NOT greater than 10.
    print("True")   # This is skipped over and never executes
else:               # Since condition evaluated to False, the 
                    # instruction in this 'block' executes..
    print("False")  # This executes

if x == 5:          # Evaluates as True. 5 is equal to 5.
    print("True")   # This executes
