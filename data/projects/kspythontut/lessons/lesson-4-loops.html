Loops

Small intro

Instructions in programming are normally executed sequentially (in the order they're written). There are, however, times where a certain block of code needs to be executed multiple times. In this instance, a loop is required.

Examples

The for loop

sample_list = [1,2,3,4]     # Create a list object. These objects can 
                            # hold multiple items in them.

for item in sample_list:    # Iterates through every item in sample_list
    print(item)             # prints 1 2 3 4

The above code goes through (iterates) the provided list.

The while loop

The while loop will execute a set of commands while the condition provided is True

i = 1                       # Creates a variable i that is
                            # initiated to a value of 1

while i < 10:               # Iterates while i is less than 10
    print(i)                # prints 1 2 3 4 5 6 7 8 9 after every
                            # loop run
    i += 1                  # Increments the value of i by 1

The above code in the while loop executes as long as the variable is less than 10.

NOTE: < is not the same as <=
