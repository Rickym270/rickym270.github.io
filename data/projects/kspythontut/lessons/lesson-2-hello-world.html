Lesson 2

Hello World!

Extremely short but remember, this is an intro still

Create project

Start up PyCharm and create a new Project and name it PythonTutorial

Create file and code!

  1. Once the project is created, make a new file and title it, helloworld.py
  2. On line 1, type the following: 
    print("Hello World!")
  3. On the taskbar, find Run then click Run.
  4. A window should appear and it should say "Hello World"

You've just built your first python program. It's small and simple but baby steps.
